# Saida Image Maker 사용 설명서

이 문서는 `image-upscaler-pro` 웹앱의 주요 기능과 사용 방법을 안내합니다.

## 1. 시작하기

1) 메인 화면에서 원하는 도구를 선택합니다.  
2) 이미지 업로드 후 편집 옵션을 조정합니다.  
3) 미리보기를 확인하고 다운로드합니다.

### 빠른 선택 가이드
- 사진을 선명하게 키우기: `/upscale`
- 용량을 줄이기: `/tools/compress`
- 크기를 정확히 맞추기: `/tools/resize`
- 포맷을 바꾸기: `/tools/convert`
- 개인정보/배경 흐림 처리: `/tools/blur`
- 원하는 부분만 잘라내기: `/tools/crop`
- 방향 돌리기/반전: `/tools/rotate`
- 워터마크 만들기: `/tools/watermark-maker`

## 2. 공통 사항

- 지원 포맷: PNG, JPG/JPEG, WebP.
- 모든 처리 과정은 브라우저 로컬에서 실행됩니다. (서버에 저장하지 않음)
- 대용량 이미지는 처리 시간이 길어질 수 있습니다.
- 일부 클라우드 업로드 버튼은 현재 준비 중입니다.
- 투명 배경은 JPG 미지원이므로 PNG/WebP를 사용하세요.
- 속도를 위해 Chrome/Edge 사용을 권장합니다(GPU 옵션).

## 3. AI 업스케일 (AI 업스케일 & 화질 개선)

경로: `/upscale`

### 추천 설정(초보자용)
- 사진: 2x + GPU 가속 ON
- 일러스트: 2x
- 텍스트/스크린샷: 2x
- 다운로드는 PNG/WebP 추천

### 업로드
- PNG/JPG/WebP 이미지를 업로드합니다.
- 업로드가 끝나면 자동으로 에디터로 이동합니다.

### 에디터 사용법
- 보정 모드: 사진 / 일러스트 / 텍스트.
- 배율 선택: 2x 또는 4x.
- 추가 옵션: 얼굴 상세 보정, GPU 가속.
- 우측 패널에서 업스케일 버튼을 눌러 처리합니다.
- 미리보기 모드에서 다운로드도 가능합니다.

### 다운로드
- 포맷: PNG / WebP / JPG.
- 처리 완료 후 다운로드 버튼을 눌러 저장합니다.
- 무료 사용자는 하루 최대 5회 다운로드 제한이 있습니다.
- 다음 날 다시 이용하면 제한이 초기화됩니다.

## 4. 이미지 압축

경로: `/tools/compress`

### 사용법
- 업로드 후 에디터에서 품질 슬라이더로 압축률을 조정합니다.
- "희망 파일 크기(KB)" 모드를 켜면 목표 용량으로 맞출 수 있습니다.
- 다운로드 포맷: JPG / PNG / WebP.
팁: 용량 우선이면 "희망 파일 크기(KB)" 추천.

## 5. 이미지 크기 조정

경로: `/tools/resize`

### 사용법
- 가로/세로 픽셀을 직접 입력합니다.
- "비율 고정"을 켜면 가로·세로 비율이 유지됩니다.
- 프리셋(Full HD, Mobile, Square, Half Size) 사용 가능.
팁: 비율 고정 후 프리셋으로 빠르게 맞추세요.

## 6. 이미지 포맷 변환

경로: `/tools/convert`

### 사용법
- PNG / JPG / WebP 중 포맷 선택.
- JPG/WebP는 품질 슬라이더 조정 가능.
- 변환된 이미지 다운로드.
팁: 투명 배경은 PNG/WebP가 안전합니다.

## 7. 이미지 블러 처리

경로: `/tools/blur`

### 사용법
- 마우스로 드래그하여 블러 영역을 추가합니다.
- 영역별로 흐림 강도와 모서리 둥글기를 조정합니다.
- 필요 없는 영역은 삭제 가능합니다.
- "블러 적용 후 다운로드"로 저장합니다.
팁: 강도는 낮게 시작해 천천히 올리세요.

## 8. 이미지 잘라내기

경로: `/tools/crop`

### 사용법
- 비율 프리셋 선택: 1:1, 16:9, 4:3, Custom.
- Custom은 중앙 80% 기준으로 자동 크롭됩니다.
- "크롭된 이미지 다운로드"로 저장합니다.
팁: SNS 업로드는 1:1 또는 16:9 추천.

## 9. 이미지 회전

경로: `/tools/rotate`

### 사용법
- -90° / +90° 회전 또는 180° 반전 선택.
- "초기화"로 원본 방향 복구.
- "회전된 이미지 다운로드"로 저장합니다.
팁: EXIF 문제면 90도 회전을 먼저 시도.

## 10. 워터마크 만들기

경로: `/tools/watermark-maker`

### 사용법
- 텍스트/로고 워터마크 선택 가능.
- 타일링 모드로 반복 워터마크 생성.
- 투명도, 크기, 회전, 간격 등을 조정합니다.
- 다운로드 포맷: PNG / WebP / JPG.
- 캔버스 사이즈는 1024x1024 고정입니다.
팁: 투명도 30~60%가 무난합니다.

## 11. 문제 해결

- 업로드가 안 될 때: 파일 포맷과 크기를 확인하세요.
- 결과가 느릴 때: 브라우저 탭을 줄이고 다시 시도하세요.
- 다운로드 실패 시: 새로고침 후 다시 시도하세요.
- 증상: 결과가 뭉개짐/노이즈 → 해결: 2x + Fidelity 낮게.
- 증상: 투명 배경이 사라짐 → 해결: PNG/WebP로 다운로드.
- 증상: 모바일/저사양에서 느림/멈춤 → 해결: 작은 이미지로 시도.
