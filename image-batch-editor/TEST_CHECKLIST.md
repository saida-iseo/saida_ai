# 🧪 Image Batch Editor - 수동 테스트 체크리스트

## 테스트 환경 준비

### 테스트 이미지 준비
- [ ] JPEG 파일 (3-5개, 다양한 크기)
- [ ] PNG 파일 (투명 배경 포함)
- [ ] WebP 파일
- [ ] 큰 이미지 (5000x5000 이상)
- [ ] 작은 이미지 (100x100 이하)
- [ ] EXIF 메타데이터 포함 이미지
- [ ] 다양한 비율 이미지 (1:1, 4:3, 16:9)

---

## 1. 파일 선택 기능

### 1.1 파일 선택
- [ ] "Add Files" 버튼 클릭 시 파일 선택 다이얼로그 열림
- [ ] 단일 파일 선택 가능
- [ ] 다중 파일 선택 가능 (Shift/Ctrl 클릭)
- [ ] 선택한 파일이 좌측 리스트에 표시됨
- [ ] 썸네일, 파일명, 크기, 해상도 정보 표시

### 1.2 폴더 선택
- [ ] "Add Folder" 버튼 클릭 시 폴더 선택 다이얼로그 열림
- [ ] 폴더 선택 시 이미지 파일만 필터링되어 추가됨
- [ ] "Include subfolders" 옵션 활성화 시 하위 폴더도 스캔
- [ ] "Include subfolders" 옵션 비활성화 시 현재 폴더만 스캔
- [ ] 지원하지 않는 포맷은 자동으로 스킵됨

### 1.3 파일 관리
- [ ] "Clear" 버튼으로 모든 파일 제거
- [ ] 파일 선택 시 미리보기 영역에 표시
- [ ] 처리 중에는 파일 추가/제거 비활성화

---

## 2. 출력 설정

### 2.1 출력 폴더
- [ ] "Output Folder" 버튼 클릭 시 폴더 선택 다이얼로그 열림
- [ ] 선택한 폴더 경로가 상단에 표시됨
- [ ] 존재하지 않는 폴더 선택 시 자동 생성

### 2.2 덮어쓰기 옵션

- [ ] 덮어쓰기 비활성화 시 출력 폴더 필수

## 3. 이미지 처리 기능

### 3.1 리사이즈

#### Longest-side 모드
- [ ] Longest-side 값 설정 (예: 1920)
- [ ] 가로가 긴 이미지: 가로가 1920으로 조정, 세로는 비율 유지
- [ ] 세로가 긴 이미지: 세로가 1920으로 조정, 가로는 비율 유지
- [ ] 정사각형 이미지: 양쪽 모두 1920으로 조정

#### Fixed 모드
- [ ] Width, Height 값 설정
- [ ] "Keep aspect ratio" 활성화 시 비율 유지
- [ ] "Keep aspect ratio" 비활성화 시 강제 리사이즈
- [ ] Width만 설정 시 세로는 비율에 맞춰 자동 계산
- [ ] Height만 설정 시 가로는 비율에 맞춰 자동 계산

### 3.2 압축

#### 품질 기반
- [ ] Quality 슬라이더 조정 (1-100%)
- [ ] Quality 100%: 최대 품질
- [ ] Quality 50%: 중간 품질
- [ ] Quality 10%: 낮은 품질
- [ ] JPEG: 파일 크기 감소 확인
- [ ] PNG: 압축 효과 확인

#### 목표 용량 기반
- [ ] Target Size (KB) 입력 (예: 500KB)
- [ ] 출력 파일이 목표 용량에 근접함
- [ ] 10회 반복 후 최적 품질 적용
- [ ] 매우 작은 목표 용량 (10KB)에서도 동작

### 3.3 크롭

#### 프리셋 크롭
- [ ] "Enable crop" 체크
- [ ] 1:1 (Square) 선택 시 정사각형 크롭 (중앙 기준)
- [ ] 4:3 선택 시 4:3 비율 크롭
- [ ] 16:9 선택 시 16:9 비율 크롭

#### 자유 크롭
- [ ] "Free" 선택
- [ ] X, Y, Width, Height 값 입력
- [ ] 좌표가 이미지 범위를 벗어나면 에러 처리

### 3.4 회전
- [ ] 0°: 회전 없음
- [ ] 90°: 시계방향 90도 회전
- [ ] 180°: 180도 회전
- [ ] 270°: 반시계방향 90도 회전

### 3.5 포맷 변환

#### JPEG
- [ ] PNG → JPEG 변환
- [ ] WebP → JPEG 변환
- [ ] 투명 배경이 있는 PNG → JPEG 시 배경색 적용
- [ ] 배경색 선택 (흰색, 검정, 커스텀 색상)

#### PNG
- [ ] JPEG → PNG 변환
- [ ] WebP → PNG 변환

#### WebP
- [ ] JPEG → WebP 변환
- [ ] PNG → WebP 변환
- [ ] 품질 조정 적용

### 3.6 메타데이터
- [ ] "Strip metadata (EXIF)" 체크
- [ ] EXIF 정보가 제거됨 (확인: ExifTool 등)
- [ ] GPS, 카메라 정보 등 모두 제거

## 4. 파일명 규칙

### 4.1 원본명 유지
- [ ] "Keep original name" 활성화
- [ ] Prefix 추가 (예: "edited_")

- [ ] Zero Padding 설정 (예: 3 → 001, 002, ...)

### 4.3 파일명 충돌
- [ ] 동일한 파일명 존재 시 자동으로 "_1", "_2" 추가
- [ ] 덮어쓰기 옵션 활성화 시 무시

## 5. 배치 처리

### 5.1 처리 시작

- [ ] 파일이 없을 때 에러 메시지
- [ ] 출력 폴더가 없을 때 (덮어쓰기 OFF) 에러 메시지

### 5.2 진행률
- [ ] 전체 진행률 표시 (예: 5/10)
- [ ] 퍼센트 표시 (예: 50%)

### 5.3 취소
- [ ] "Cancel" 버튼 클릭
- [ ] 처리 즉시 중단
- [ ] 이미 처리된 파일은 유지
- [ ] 처리 중이던 파일은 불완전하게 저장될 수 있음

### 5.4 완료
